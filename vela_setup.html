Konfigurera databasen

Vela använder Supabase för att lagra patientdata säkert i EU. Skapa ett gratis konto och klistra in dina nycklar.

1
Skapa Supabase-projekt
Gå till supabase.com → New project → välj region Frankfurt (eu-central-1).
Öppna Supabase →
2
Kopiera dina nycklar
Settings → API → kopiera Project URL och anon public key.
3
Skapa databastabellen
Kör detta SQL i Supabase → SQL Editor → New query:
create table if not exists patients ( id text primary key, data jsonb, updated_at timestamptz default now() ); create table if not exists pass_log ( id bigserial primary key, visit_id text not null, pain numeric, done text, logged_at timestamptz default now() ); -- RLS alter table patients enable row level security; alter table pass_log enable row level security; create policy "anon read patients" on patients for select using (true); create policy "anon update patients" on patients for update using (true); create policy "anon insert patients" on patients for insert with check (true); create policy "anon insert pass_log" on pass_log for insert with check (true); create policy "anon read pass_log" on pass_log for select using (true);

Koppla AI-motorn

Vela använder Anthropic Claude för att generera rehabplaner och journaler. Du behöver en API-nyckel.

1
Skapa Anthropic-konto
Gå till console.anthropic.com och logga in.
Öppna Anthropic →
2
Hämta API-nyckel
API Keys → Create Key → kopiera nyckeln (börjar med sk-ant-).
Nyckeln sparas lokalt i din webbläsare — aldrig på vår server.
